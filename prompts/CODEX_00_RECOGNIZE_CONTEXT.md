1. Inspecciona la estructura actual del repositorio.

3. Resume en tus palabras qué es UFO Predictor.
4. Resume el stack objetivo.
5. Resume qué NO debe incluirse en el primer prototipo.
6. Resume las páginas que debe tener el primer prototipo.
7. Resume los componentes principales que deberán crearse.
8. Resume las integraciones que deben quedar solo como skeleton/TODO.
9. Detecta posibles contradicciones o vacíos en el contexto.
10. No crees archivos de código todavía.
11. No instales dependencias todavía.
12. No modifiques el repo todavía.

Entrega tu respuesta con estas secciones:

- Estado actual del repo.
- Contexto entendido.
- Stack entendido.
- Alcance del primer prototipo.
- Exclusiones importantes.
- Riesgos o dudas detectadas.
- Siguiente prompt recomendado para crear el prototipo.
